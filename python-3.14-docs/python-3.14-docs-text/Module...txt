
------------------------awk----------------------------------


| Task                   | Command                    |                                
| ---------------------- | -------------------------- |
| Print all lines        | `awk '{print}' file`       |
| Print full line        | `awk '{print $0}' file`    |
| Print 1st column       | `awk '{print $1}' file`    |
| Print multiple columns | `awk '{print $1,$3}' file` |
| Print line number      | `awk '{print NR,$0}' file` |
| Print last column      | `awk '{print $NF}' file`   |
| Task         | Command                             |
| ------------ | ----------------------------------- |
| Print line 5 | `awk 'NR==5 {print}' file`          |
| Print range  | `awk 'NR>=2 && NR<=6 {print}' file` |
| Even lines   | `awk 'NR%2==0 {print}' file`        |
| Odd lines    | `awk 'NR%2!=0 {print}' file`        |
| Task          | Command                       |
| ------------- | ----------------------------- |
| Match pattern | `awk '/ERROR/ {print}' file`  |
| Not match     | `awk '!/ERROR/ {print}' file` |
| Pattern range | `awk '/A/,/B/' file`          |
| Task                 | Command                         |
| -------------------- | ------------------------------- |
| Count lines          | `awk 'END{print NR}' file`      |
| Count using variable | `awk '{c++} END{print c}' file` |
| Task            | Command                                     |
| --------------- | ------------------------------------------- |
| Sum column      | `awk '{sum+=$3} END{print sum}' file`       |
| Conditional sum | `awk '$3>50 {sum+=$3} END{print sum}' file` |
| Question          | Answer        |
| ----------------- | ------------- |
| Print 1st column  | `{print $1}`  |
| Print last field  | `{print $NF}` |
| Line number       | `NR`          |
| Number of fields  | `NF`          |
| Default action    | `print`       |
| Default separator | space         |

---------------------------grep---------------------------------------
|          Command         |       Description             |          Example               |
| ------------------------ | ----------------------------- | ------------------------------ |
| `grep "pattern" file`    | Search for pattern in a file  | `grep "error" logfile.txt`     |
| `grep -i "pattern" file` | Case-insensitive search       | `grep -i "warning" syslog`     |
| `grep -r "pattern" dir/` | Recursive search in directory | `grep -r "fail" /var/log`      |
| `grep -n "pattern" file` | Show line numbers             | `grep -n "timeout" config.txt` |
| `grep -v "pattern" file` | Invert match (exclude)        | `grep -v "INFO" app.log`       |
| `grep -w "word" file`    | Match whole word only         | `grep -w "cat" animals.txt`    |
| `grep -l "pattern" *`    | Show file names only          | `grep -l "main" *.c`           |
| `grep -c "pattern" file` | Count matching lines          | `grep -c "404" access.log`     |
|                                                                                             |
| ---------------------------- | --------------------------- | ------------------------------ |
| `grep -A NUM "pattern" file` | Show NUM lines after match  | `grep -A 3 "error" log.txt`    |
| `grep -B NUM "pattern" file` | Show NUM lines before match | `grep -B 2 "panic" kernel.log` |
| `grep -C NUM "pattern" file` | Show before & after         | `grep -C 2 "failed" syslog`    |
|                                                                                               |                
| ----------------------- | ----------------------- | ------------------------ | -------------- |
| `grep -E "regex" file`  | Extended regex (egrep)  | `grep -E "cat            | dog" pets.txt` |
| `grep -F "string" file` | Fixed string (no regex) | `grep -F "a*b" file.txt` |                |

import requests

BASE_URL = "https://jsonplaceholder.typicode.com/posts"

# ---------------- CREATE ----------------
create_data = {
    "title": "Hello",
    "body": "Learning CRUD",
    "userId": 1,
    "id":1
}

create_res = requests.post(BASE_URL, json=create_data)
print("\nCREATE (POST)")
print(create_res.json())


# ---------------- READ ----------------
read_res = requests.get(f"{BASE_URL}/1")
print("\nREAD (GET)")
print(read_res.json())


# ---------------- UPDATE (PUT - full update) ----------------
update_data = {
    "title": "Hello - Updated",
    "body": "Learning CRUD - Updated using PUT",
    "userId": 1
}

update_res = requests.put(f"{BASE_URL}/1", json=update_data)
print("\nUPDATE (PUT)")
print(update_res.json())


# ---------------- PATCH (partial update) ----------------
patch_data = {
    "title": "Hello - Patched"
}

patch_res = requests.patch(f"{BASE_URL}/1", json=patch_data)
print("\nPATCH (Partial Update)")
print(patch_res.json())


# ---------------- DELETE ----------------
delete_res = requests.delete(f"{BASE_URL}/1")
print("\nDELETE")
print("Status Code:", delete_res.status_code)
